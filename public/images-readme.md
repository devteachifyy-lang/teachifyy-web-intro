## How to Add Images

1. Use https://unsplash.com or https://pexels.com for free stock photos
2. Use https://ui-avatars.com for quick avatar generation
3. Recommended dimensions:
   - Course thumbnails: 1200x675px (16:9 aspect ratio)
   - Blog thumbnails: 1200x675px (16:9 aspect ratio)
   - Avatars: 400x400px (square)
   - Founder photo: 800x800px (square)
   - OG image: 1200x630px

For now, you can use placeholder services like:
- `https://placehold.co/1200x675/1a1a1a/FFC107?text=Course+Thumbnail`
- `https://ui-avatars.com/api/?name=John+Doe&size=400&background=FFC107&color=0A0A0A`
